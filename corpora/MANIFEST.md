# Source Corpora Manifest

This file describes the source materials required by the project as specified in PROJECT.md §5. Each corpus must be addressable at the verse or section level and carry explicit version and edition metadata.

## Current Status

This is a placeholder manifest. The actual corpora will be assembled progressively during Phase 0.

## Required Source Categories

### Scripture, Original Languages

1. **Hebrew Bible**
   - Edition: Biblia Hebraica Stuttgartensia (or BHQ where available)
   - Status: Not yet acquired
   - Notes: Main source for Old Testament in original Hebrew/Aramaic

2. **Septuagint**
   - Edition: Rahlfs or Göttingen LXX
   - Status: Not yet acquired
   - Notes: Greek translation of the Hebrew Bible for comparative analysis

3. **Greek New Testament**
   - Edition: NA28 ( Nestle-Aland 28th edition)
   - Status: Not yet acquired
   - Notes: Critical text for the New Testament

4. **Byzantine/Majority Text**
   - Edition: To be determined
   - Status: Not yet acquired
   - Notes: Alternative text base for New Testament

5. **Textus Receptus**
   - Edition: To be determined
   - Status: Not yet acquired
   - Notes: Traditional text base for some Protestant traditions

6. **Peshitta**
   - Edition: To be determined
   - Status: Not yet acquired
   - Notes: Syriac witness to the biblical texts

7. **Vulgate**
   - Edition: To be determined
   - Status: Not yet acquired
   - Notes: Latin tradition text

## Licensing Notes

Each corpus item will include explicit licensing information. Where licensing restrictions prevent direct inclusion in the repository, we will provide:
1. Machine-readable addressing scheme
2. Clear instructions for obtaining the source material
3. Version tags for reproducibility

## Acquisition Strategy

Progressive acquisition prioritized by:
1. Core original language texts needed for initial debates
2. Primary confessional documents for major traditions
3. Representative translations for comparative analysis
4. Secondary theological works as needed for debate seeding

## Addressing Scheme

All corpora will be addressable at the verse or section level using standard biblical referencing:
- Book, chapter, verse for biblical texts (e.g., "Romans 3:24")
- Section identifiers for confessional/historical documents (e.g., "Westminster Confession 3.11.2")
